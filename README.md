# (Yet Another) iTunes Artwork Grabber

Get high-res artwork for apps, music, books, movies, TV shows, podcasts and more, all types supported.

## Supported Types

- iOS app icons
- OS X app icons
- Music album covers
- Book covers
- Movie posters
- TV show covers
- Podcast covers
- iTunes U resource covers
- Audiobook covers

## Unsupported Types

- Music video thumbnails
